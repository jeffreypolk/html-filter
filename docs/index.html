html-filter

JavaScript to filter html elements

View project on GitHub

Overview

Provides simple filtering of html elements based on user input.

Dependencies

The project was built on the following:

  • jQuery 3.1.1 - while not tested, I'm sure this works on older versions as well.
  • highlight by kasperp to provide keyword highlighting. This is only needed if using the highlight = true option.

Setup

To use html-filter in your project, include the following:

  • ps-filter.css
  • ps-filter.min.js

Markup

The filtering works by attaching to the keyup event on an input element. Matching HTML elements (via css class) will be tested by comparing the input to the text() of the element. Using text() will do a deep match, which is good if you have multiple <div> or <span> elements as children of the matching element. But, this can be a drawback if the input is "div" because it will match all elements.

No JavaScript is required for filtering. Just add the ps-filter class to the input control, and the ps-filter-item to each HTML element that should be filtered. The necessary events and code will be automatically added during the $(document).ready() sequence.

Multiple filters can exist on a page by defining the target filter css class via data attribute.

Options

Filtering can also be initiated via JavaScript. The following are the default options:

Name Default Description
filterClass ps-filter-item A css class used by each HTML element that should be filtered.
highlight false A boolean controlling if text highlighting should be used within matched elements.

Methods

There are no methods available for this plugin.

Other Notes

  • The filtering mechanism adds a hidden css class to an element when it doesn't match the input. This class applies display:none; to the element.
  • If filtering rows in a bootstrap table with the table-striped class, rows will be re-striped after filtering. It does this by removing the table-striped class from the table. It then applies the ps-filter-stripe class to every odd row still visible. This class uses a color of #f9f9f9. You may override this class if using a different color for your table.
  • If using highlighting, the color is defined by the highlight css class. The default color is yellow. You may override this class if you wish to use another color.
